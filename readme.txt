Survey website
This website is designed to collect user survey data, analyze it in-depth, 
and generate insightful statistics. It empowers you to understand participant responses,
uncover trends, and make informed decisions based on your audience's feedback.



Prerequisites:

XAMPP (https://www.apachefriends.org/download.html)
Web browser
Installation Stages:

1. Setting Up XAMPP
Download and install XAMPP for your operating system.
Start the Apache and MySQL modules in the XAMPP control panel.

2. Extracting Website Files
Extract the contents of this website package into your XAMPP installation's "htdocs" folder (usually located at C:\xampp\htdocs).

3. Importing Tables
Access PHPMyAdmin through your web browser (usually at http://localhost/phpmyadmin).
Create a new database (e.g., "my_website").
Navigate to the "tables" folder within the website directory.
Select all .sql files and import them into the newly created database.

4. Accessing the Website
Open your web browser and go to http://localhost/index.php.
The website should now be up and running.
Troubleshooting:

If you encounter errors, ensure that Apache and MySQL are running in XAMPP.
Check for typos in database names and file paths.
Refer to the error logs for more specific troubleshooting information.
Additional Notes:

The "index.php" file is the starting point of the website.
Customize and modify the website as needed.
For further assistance, consult XAMPP and PHPMyAdmin documentation.
